# План оптимизации сайта по продаже цветов
## 1. Для чего нужна оптимизация?
- Улучшить пользовательский опыт и ускорить загрузку сайта
- Повысить конверсию в заказ
- Снизить нагрузку на сервер и расходы на обслуживание
- Улучшить позиции сайта в поисковых системах

## 2. Что и как будем оптимизировать?
**2.1.Фронтенд** 

1. Минификация CSS и JavaScript
2. Оптимизация изображений букетов: WebP, lazy loading
3. Кэширование статических файлов в браузере
4. Адаптация сайта под мобильные устройства

**2.2. Каталог и карточки товаров**

1. Оптимизация загрузки фотографий букетов
2. Быстрая подгрузка каталога и фильтров
3. Упрощение структуры карточек товара
4. Сжатие изображений без потери качества

**2.3. Бэкенд**

1. Кэширование частых запросов
2. Оптимизация базы данных для быстрого поиска товаров
3. Ускорение обработки заказов и заявок
4. Асинхронная обработка уведомлений, аналитики и обновления остатков

**2.4. SEO**

1. Семантическая вёрстка страниц каталога и карточек товаров
2. Генерация уникальных страниц по категориям: букеты на день рождения, цветы для мамы, доставка цветов и другие
3. Оптимизация мета-тегов и заголовков
4. Улучшение структуры URL и внутренней перелинковки

## 3. Ожидаемые результаты

1. Сокращение времени загрузки основных страниц
2. Более быстрый переход от каталога к оформлению заказа
3. Снижение числа отказов
4. Рост конверсии в заявку или покупку
5. Улучшение видимости сайта в поисковых системах

## 4. Источники данных

1. Исследования Google о влиянии скорости загрузки на конверсию
2. Практики оптимизации интернет-магазинов
3. Рекомендации по SEO для карточек товаров и каталогов
4. Подходы к оптимизации изображений и мобильной производительности
