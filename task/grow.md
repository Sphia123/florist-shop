# План развития сайта по продаже цветов (Roadmap)
## 1. Краткосрочные улучшения (3–6 месяцев)
- Каталог товаров: заменить тестовые позиции на полноценный ассортимент букетов и композиций с актуальными фото, описаниями и ценами
- Бэкенд и база данных: разработать серверную часть для хранения пользователей, заказов, каталога, статусов доставки и истории покупок
- Регистрация и вход: подключить авторизацию через email, номер телефона или социальные сети
- Платёжная интеграция: подключить платёжный сервис для приёма онлайн-оплаты
- Личный кабинет: добавить профиль пользователя с историей заказов, адресами доставки и избранными товарами
- Мобильная адаптация: доработать интерфейс для удобного оформления заказов со смартфона

## 2. Среднесрочные улучшения (6–12 месяцев)

Расширение функций заказа:
- Выбор даты и времени доставки
- Добавление открытки к букету
- Заказ в один клик
- Повторный заказ из истории покупок
Персонализация:
- Рекомендации на основе просмотренных и купленных букетов
- Подборки по поводу, получателю и бюджету
- Блоки «Похожие букеты» и «Часто выбирают»
Дополнительные каналы:
- Telegram-бот для быстрого оформления заказа
- Интеграция с социальными сетями для публикации акций и новинок

## 3. Долгосрочные улучшения (1–2 года)

- **Конструктор букета**: возможность собрать индивидуальную композицию из выбранных цветов
- **Подписка на регулярную доставку**: еженедельные или ежемесячные букеты по подписке
- **Корпоративные заказы: отдельные** условия для компаний и массовых заказов
- **Партнёрства**: сотрудничество с кондитерскими, магазинами подарков и праздничными сервисами
- **Программа** лояльности: бонусы, скидки, накопительная система и специальные предложения для постоянных клиентов
