# BloomFlow

Учебный проект — интернет-магазин цветов и букетов

Сайт позволяет просматривать каталог букетов, добавлять товары в корзину, оформлять подписку, а также переходить на страницы входа и регистрации. Все данные хранятся локально, внешние серверы не используются

## Структура репозитория

- `flotist/` — исходный код проекта (страницы, стили, скрипты)
- `task/` — документация и анализ проекта

## Запуск

```bash
cd flotist
npm install
npm run dev
